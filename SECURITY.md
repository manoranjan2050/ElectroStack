# Security Policy

ElectroStack manages local services and credentials. Treat every issue involving credentials, update verification, firewall state, local privilege boundaries, or service command execution as security-sensitive.

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting

Open a private security advisory on GitHub or contact the project maintainers. Include reproduction steps, affected version, expected impact, and suggested remediation when available.
